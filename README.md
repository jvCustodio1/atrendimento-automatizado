# Assistente Inteligente para Atendimento Automatizado - NR Caçambas 🚀

[cite_start]Projeto de Prática Curricular Extensionista desenvolvido no módulo de **Desenvolvimento de Inteligência Artificial** do curso de Análise e Desenvolvimento de Sistemas[cite: 9].

[cite_start]O objetivo é resolver um problema real da empresa **NR Caçambas**: a sobrecarga no atendimento via WhatsApp nos horários de pico, a lentidão nas respostas e o risco de perda de informações devido ao registro manual de pedidos[cite: 11, 12].

---

## 📋 Sobre o Projeto

[cite_start]O sistema consiste em um protótipo funcional de assistente virtual inteligente que simula a interface do WhatsApp[cite: 14]. [cite_start]Ele automatiza o primeiro contato com o cliente, responde dúvidas frequentes e estrutura os dados para a criação de pedidos de forma organizada[cite: 13, 57].

### Principais Funcionalidades
* [cite_start]**Interface Web Responsiva:** Tela simulando o layout do WhatsApp para uma demonstração realista[cite: 14, 54].
* [cite_start]**Atendimento Automatizado com IA:** Interpretação de mensagens e respostas inteligentes utilizando APIs de IA[cite: 61, 80].
* [cite_start]**Triagem e Coleta de Dados:** Coleta automática de informações essenciais para a locação (tamanhos, valores, prazos e endereço)[cite: 30, 57].
* [cite_start]**Base de Dados Integrada:** Registro automatizado dos pedidos em uma planilha do Google Sheets para controle interno da empresa[cite: 71, 79].

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* [cite_start]**Front-end:** HTML5, CSS3 e JavaScript [cite: 78]
* [cite_start]**Inteligência Artificial:** Integração com APIs de IA para processamento de linguagem natural [cite: 80]
* [cite_start]**Banco de Dados:** Google Sheets API (como persistência simples e acessível) [cite: 71, 79]
* [cite_start]**Ambiente de Desenvolvimento:** VS Code e GitHub [cite: 81, 82]
