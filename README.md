# Assistente Inteligente para Atendimento Automatizado - NR Caçambas

Projeto de Prática Curricular Extensionista desenvolvido no módulo de Desenvolvimento de Inteligência Artificial do curso de Análise e Desenvolvimento de Sistemas.

O objetivo é resolver um problema real da empresa NR Caçambas: a sobrecarga no atendimento via WhatsApp nos horários de maior movimento, a lentidão nas respostas e o risco de perda de informações devido ao registro manual de pedidos.

---

## Sobre o Projeto

O sistema consiste em um protótipo funcional de assistente virtual inteligente que simula a interface do WhatsApp. Ele automatiza o primeiro contato com o cliente, responde dúvidas frequentes e estrutura os dados para a criação de pedidos de forma organizada.

### Principais Funcionalidades
* **Interface Web Responsiva:** Tela simulando o layout do WhatsApp para uma demonstração realista.
* **Atendimento Automatizado com IA:** Interpretação de mensagens e respostas inteligentes utilizando APIs de IA.
* **Triagem e Coleta de Dados:** Coleta automática de informações essenciais para o registro de pedidos.
* **Base de Dados Integrada:** Registro automatizado dos pedidos em uma planilha do Google Sheets para controle interno da empresa.

---

## Tecnologias e Ferramentas Utilizadas

* **Front-end:** HTML, CSS e JavaScript
* **Inteligência Artificial:** APIs de Inteligência Artificial
* **Banco de Dados:** Google Sheets
* **Ambiente de Desenvolvimento:** Visual Studio Code e GitHub
